Introduction to Python

Python is a high-level, general-purpose programming language. Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented, and functional programming. Python consistently ranks among the most popular programming languages today.

Python was created by Guido van Rossum, who began working on Python in the late 1980s. He was the sole developer of Python from 1991 until his retirement in 2018. In January 2019, active Python core developers elected a five-member Steering Council to lead the project.

The core philosophy of the Python programming language is summarised in the document The Zen of Python, which includes elements such as:

  • Beautiful is better than ugly.
  • Explicit is better than implicit.
  • Simple is better than complex.
  • Complex is better than complicated.
  • Readability counts.
Data Types

There are a few data types in Python, mainly string, int, float, boolean.

Declaring Variables

String variable is declared using double quotes (" ") or single quotes (' ').

In this example, a string variable with a name of "name" is declared, the value being "Jason".

    name="Jason"

Int variable is declared by assigning an integer value to a variable. For example, to declare a "price" variable with the value of 10, the following code can be used: 

price=10

Float type is different from int type in the sense that float data types can store decimal point values, while int data type cannot store decimal point values. For example, to declare a "height" variable with a value of 170.5, the following code can be used: 

height=170.5

Boolean data types can only store two values, either True or False (note the capitalisation). For example, to declare an "alive" variable with the value of True, the following code can be used: 

alive= True
Printing to the console

Time for the fun part, print statements. The print() function in Python allows a user to output text enclosed within the print function to the console window.

To output "Hello World!" into the console, the following code can be used:


print("Hello World!")

The code above will produce the following output: 

Hello World!

We could also store the value "Hello World!" using a string variable, and then print to the console window. It will return the same output as before.


greeting="Hello World!" 
print(greeting)
String Concatenation

Now suppose we want to combine several variables into a single print() function. Consider the example below:


name="Jason"
age=17
height: 170

What should we do if we want the output of a print() function to be "My name is Jason, I am 17 years old and my height is 170 cm."? This is when string concatenation comes into play.

We could do something like this: 


name="Jason"
age=17
height: 170
print("My name is "+name+", I am "+age+" years old and my height is "+height+" cm.")

However, it is not very readable and takes a lot of work. Instead, we can use an "f string", like so.


name="Jason"
age=17
height: 170
print(f"My name is {name}, I am {age} years old and my height is {height} cm.")

This will return: 


My name is Jason, I am 17 years old and my height is 170 cm.
Accepting User Input

In Python, to accept user input, the input() function is used. Do note that the input() function always returns a string value.

For example, to prompt a user to enter their name, we can use the following code: 


name = input("Enter your name: ")

The response given by the user will be stored in the "name" variable.
